## How to use
Add the JitPack repository to your build.gradle:
```gradle
repositories {
    maven { url "https://jitpack.io" }
}
```

Add the dependency:
```gradle
dependencies {
    compile 'com.github.nekocode:ToolbarIndicator:{latest version}'
}
```
